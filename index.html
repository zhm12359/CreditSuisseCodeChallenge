About

PointNoded is a web application that visualizes the trades between internal sectors of Credit Suisse. The application was developed by Krishna Thiyagarajan, Herbert Li and Hanming Zeng. The web application uses JSON formatted data with trade information between sectors and arranges them in an orderly manner.

PointNoded was designed with the fact the users using this will be utlizing this data to view it quickly and effectively so that they can make good decisions on trades. This was accomplished using the vis.js runtime environment along with the javascript language.
